# RemnaWave Bedolaga Support Bot 🎫

Бот технической поддержки для VPN-сервисов, построенных на панели **RemnaWave**. 
Позволяет обрабатывать запросы пользователей в структурированном виде через Telegram Topics.

## ✨ Особенности
- **Интеграция с БД продаж:** При открытии тикета админ сразу видит статус подписки, баланс и остаток трафика пользователя.
- **Система тикетов:** Каждое обращение — это отдельный топик в вашей группе поддержки.
- **Безопасность:** Все чувствительные данные вынесены в переменные окружения.
- **Умные логи:** Локальное хранение истории тикетов и системы банов (SQLite).

## 🚀 Быстрый старт

1. Склонируйте репозиторий.
2. Скопируйте `.env.example` в `.env` и заполните свои данные.
3. Запустите бота:
   ```bash
   docker compose up -d --build
   ```

## 🛠 Требования
- Docker & Docker Compose
- Установленная панель RemnaWave (бот подключается к её базе данных через внутреннюю сеть Docker).
